# AI-Powered Sentiment Analysis System
An AI-powered Sentiment Analysis web application developed using Machine Learning and Natural Language Processing (NLP).

## Features
- Positive / Negative / Neutral sentiment prediction
- Real-time text analysis
- Confidence score visualization
- Animated modern frontend UI
- NLP preprocessing pipeline
- Machine Learning integration

## Technologies Used
- Python
- Flask
- Scikit-learn
- NLP
- Multinomial Naive Bayes
- HTML
- CSS
- JavaScript
- Pandas
- NumPy

## Machine Learning Workflow
1. Text Preprocessing
2. Lemmatization
3. TF-IDF Vectorization
4. Sentiment Classification using MultinomialNB

## How to Run

```bash
pip install -r requirements.txt
python train_model.py
python app.py
```

Open:
http://127.0.0.1:5000
